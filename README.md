# 🌳 Zen Garden Focus & Wellness

A focus timer app where a tree grows as you work.

## Features
- **Growth Timer** — Tree grows through 6 stages as your session progresses
- **To-Do List** — Add and complete focus tasks
- **Profile** — Track sessions, streaks, and trophies

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```
